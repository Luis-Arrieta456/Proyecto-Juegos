== *Problema*

Actualmente, existe la necesidad de desarrollar aplicaciones interactivas que permitan a los usuarios experimentar la responsabilidad de cuidar a un animal sin las limitaciones físicas o de espacio. El problema consiste en diseñar y codificar un sistema de simulación de mascotas virtuales que gestione de forma dinámica el estado de salud, energía y felicidad de tres especies distintas (Perro, Gato y Loro).

El sistema debe resolver el desafío de la persistencia de estados y la interacción diferenciada, donde cada mascota reaccione de manera única según su especie a estímulos como la alimentación, el juego y el descanso, bajo un modelo de Programación Orientada a Objetos (POO).

=== *Requerimientos Funcionales*

* *Gestión de Persistencia y Configuración*

* El sistema debe cargar automáticamente los datos existentes de la mascota desde un archivo de persistencia en formato JSON al iniciar.
* Si no existe una mascota guardada, el sistema debe permitir al usuario seleccionar una especie (Gato, Perro o Loro) y asignar un nombre.
* El sistema debe guardar el progreso actual (niveles y estado) en el archivo JSON antes de cerrar el programa.

* *Simulación y Estados*

* El sistema debe ejecutar un ciclo continuo que permita la interacción constante hasta que el usuario decida salir.
* El programa debe generar de forma aleatoria (con una probabilidad mayor al 0.7) necesidades de comer, dormir o jugar.
* El sistema debe calcular y mostrar en tiempo real los niveles de Hambre, Felicidad y Experiencia de la mascota.

* *Interacción y Acciones*

* El sistema debe procesar comandos del usuario para alimentar, jugar, dormir o aplicar disciplina a la mascota.
* El sistema debe impedir la acción de "Jugar" si la mascota tiene 20 o menos puntos de energía, notificando al usuario sobre su cansancio.
* Al seleccionar la opción "Salir", el sistema debe guardar el progreso actual antes de finalizar la ejecución.

* *Comportamientos Específicos*

* Cada mascota debe emitir su sonido característico (ladrar, maullar o hablar) al realizar acciones de alimentación o juego.
* Debe reducir el nivel de hambre según la especie (Perro -30, Gato -25, Loro -20).
  * Jugar: Debe aumentar la felicidad en 30 puntos y consumir 25 puntos de energía.
  * Dormir: Debe restaurar la energía de la mascota al 100%.
